# Stone Paper Scissors Game v2.1

A Python-based Stone Paper Scissors game built using core Python concepts.

##  Features
- User vs Computer gameplay
- Score tracking
- Random computer choices
- Creative tie-handling system to manage draw situations
- Improved logic and structure (v2.1)

##  Concepts Used
- Python basics
- `random` module
- Loops and conditionals
- User input handling

##  How to Run
1. Install Python (latest version)
2. Run the file using any one method:
   - Double-click `main.py`
   - Open Command Prompt and run:
     ```bash
     python main.py
     ```
   - Open the file in an IDE and press **F5**
